Social Media Sentiment Analysis

  • Tech Stack: Python, HTML, CSS

My Social Media Sentiment Analysis project is designed to classify and analyze emotions expressed in social media posts, with a focus on Twitter data. Using a Logistic Regression Model, the system efficiently categorizes text into positive, negative, or neutral sentiments based on patterns in the language. The preprocessing pipeline is carefully designed to handle challenges such as sarcasm detection, negation handling, and keyword-based sentiment amplification, ensuring a more accurate classification.

For better visualization, the project utilizes Lollipop Graphs to represent sentiment distribution in a clear and concise manner, making it easy to compare sentiment frequencies. Additionally, Sunburst Charts offer a hierarchical view of sentiment breakdown, enabling deeper analysis of sentiment relationships and trends over time. This project has practical applications in brand reputation management, customer feedback analysis, public opinion tracking, and content moderation, helping businesses, researchers, and organizations gain valuable insights from social media data.
